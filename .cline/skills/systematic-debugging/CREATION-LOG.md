## Validation Approach

Tested with scenarios covering:
- normal debugging conditions
- time pressure
- repeated failed fixes
- uncertain root cause

## Outcome

The skill emphasizes evidence gathering, root-cause tracing, and single-change hypothesis testing.

## Usage Reminder

When a bug appears:
1. investigate evidence
2. trace source
3. test one hypothesis
4. implement and verify
