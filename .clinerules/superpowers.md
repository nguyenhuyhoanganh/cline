# Superpowers Rules

This project uses Superpowers skills from `.cline/skills`.

Rules:

- Before doing substantial work, load the relevant Superpowers skill.
- For vague/new feature requests, start with `superpowers:brainstorming`.
- Before implementation, create a concrete task list with `superpowers:writing-plans`.
- During implementation, apply `superpowers:test-driven-development` where feasible.
- Before finishing, run `superpowers:requesting-code-review`.
- If multiple skills could apply, prefer process skills first, then implementation skills.
